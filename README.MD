# Movie Review - React.
Repositório destinado a desafio de página de filmes.
- Página inicial: lista de filmes
- Botão para página de informações de cada filme
- Página para vizualizar informações de cada filme

## Feito com
- HTML
- CSS
- JavaScript
- Fetch API
- Vite
- React
- React Router
